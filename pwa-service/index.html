Component Library

jellicule is a self-bootstrapping, self-building, self-testing, and self-documenting web component library with a retro 80s aesthetic.

Features

  • Wireframe, outlined components
  • Neon glow effects
  • Retro 80s color palette
  • Minimal dependencies
  • Fully customizable
